Practical implementation of Virtual Lan

  1. First of all make the following diagram with the help of cisco.
  2. After that give IP to all the devices.
  3. You can do the Vlan configuration With the help of commands as well as GUI option.
  4. To implement Vlan using commands, you can use the following code:-

    5. SWITCH>EN
    SWITCH#CONF T
    SWITCH(CONFIG)#VLAN 10
    SWITCH(CONFIG-VLAN)#NAME SALES
    SWITCH(CONFIG-VLAN)#EXIT
    SWITCH(CONFIG)# INTERFACE FASTETHERNET0/2
    SWITCH(CONFIG-IF)# SWITCHPORT MODE ACCESS
    SWITCH(CONFIG-IF)# SWITCHPORT ACCESS VLAN 10

  5. When all the connection work is done then the first thing is to check your connection that is working efficiently or not.
  6. For checking that we have to ping your HR department PC with University PC then Request Time out message will be display device
  7. If not happen then make your you make a good connection
  8. When we ping the device with its own department PC then it will easily connected with that PC
