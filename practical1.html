To connect switch and 3 PC :-

  1. In the Cisco Packet Tracer, click on the PC icon, click Generic, and then click in the logical view area to add a Generic PC.
  2. Repeat the same step to add three more Generic PCs in the logical view area.
  3. To add a switch in Cisco Packet Tracer, Click The switch icon, select a switch type, such as 2950, and then add the selected switch in the logical view area.
  4. Complete the cabling :- Access the cables section and connect completely and correctly the cables. Some of the common types of cables are straight-through, cross-over, serial-DCE, Console. Once you are familiar with the types of connections, connect the devices to create the Network Topology.
  5. To connect devices in Cisco Packet Tracer, click the connection type icon, and select appropriate cable. For example, to connect PC0 to switch, select the straight-through cable, click on switch, select FastEthernet0 and then click on PC0 also select the FastEthernet0 interface.
  6. Now, add PC1 to switch using the FastEthernet0 interface. Also, add PC2 to PC5 with FastEthernet0 interface of only one switch respectively.
  7. If you have connected a wrong device to a wrong interface, you can use the DELETE option to delete a connection or device.
  8. Configure the IP addresses on all end devices. This can be done by accessing the desktop platform on each device and locating the IP configuration section. The reason for doing this is to enable the devices be on the right network.

Test connectivity :- After configuring, you will have to test connectivity by opening a command prompt window on the end devices and try pinging the address which the network operates on. If it gives you a reply, it means your network was configured correctly.
